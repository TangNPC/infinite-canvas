# 画布节点操作手册

本文档记录当前纯生图分支的画布节点主要用途和操作流程。

## 文本节点

文本节点用于保存提示词、草稿、说明文案和 AI 生成的文字结果。它既可以作为独立文本内容，也可以作为下游生成配置节点的输入。

### 编辑文本内容

- 双击文本节点内容区域，直接编辑节点里的文字。
- 选中文本节点后，点击顶部工具栏的“编辑文字”，进入文字编辑状态。
- 顶部工具栏的“缩小”和“放大”用于调整文本节点字号。

### 生成或改写文本

- 选中文本节点后，点击顶部工具栏的“编辑”，打开节点下方对话框。
- 文本节点下方对话框只用于生成或改写文本，不承担生图功能。
- 空文本节点会把生成结果回填到当前节点。
- 已有内容的文本节点会在右侧生成新的文本节点，并自动连接原节点和新节点。
- 输入内容可以手写，也可以从提示词库选择。

### 用文本节点生成图片

- 在文本节点中准备好生图提示词。
- 点击文本节点顶部工具栏的“生图”按钮。
- 系统会在文本节点右侧创建生成配置节点，并连接文本节点到生成配置节点。
- 生成配置节点读取上游文本作为提示词，用户可在配置节点中调整模型、比例、数量后执行。

## 图片节点

- 支持上传、拖入、替换和下载图片。
- 支持把图片节点保存到“我的素材”。
- 支持裁剪图片，并把裁剪结果生成为新的图片节点。
- 支持局部遮罩编辑，确认后创建“局部重绘提示词”和“局部重绘配置”节点，不自动发起请求。
- 支持图片拆分，按行列生成多个图片子节点。
- 支持浏览器端图片放大，生成 1K、2K 或 4K 新图片节点。
- 支持反推提示词、查看大图、复制提示词和自定义图片悬浮工具栏。

## 生成配置节点

- 支持文本模式和图片模式。
- 文本模式读取上游文本和图片引用后发起文本/反推提示词相关请求。
- 图片模式读取上游文本作为提示词，读取上游图片作为参考图。
- 支持模型、尺寸、质量、格式、压缩、审核、数量、失败重试和输入顺序配置。
- 执行结果会生成新的文本节点或图片节点，并保留生成参数用于重试。

## 推荐流程

1. 新建文本节点，写入图片创作想法。
2. 让 AI 优化或扩写提示词，保留原始文本方便对比。
3. 确认文本节点内容后，点击“生图”。
4. 在生成配置节点中调整模型、尺寸、数量或参考图。
5. 手动执行生成，并把满意结果保存到“我的素材”或继续接入下游节点。
