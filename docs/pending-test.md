# 待测试

## v0.3.3 上游合并候选

- 本次只创建合并分支、更新版本文件和文档，不提交、不打 tag、不推送、不创建 Release；验收重点见“v0.3.3 上游合并差异”文档。
- 视频创作台需要重点验证 Seedance Agent Plan 文生视频、图片参考、视频参考、音频参考、竖屏/横屏比例、清晰度、时长、生成声音、水印、任务完成状态、视频播放、下载到本地、同步到 R2、单条删除和账号视频历史同步。
- Seedance 请求需要确认默认模型 `doubao-seedance-2.0`，创建任务走 `POST /contents/generations/tasks`，轮询走 `GET /contents/generations/tasks/{id}`，完成后读取 `content.video_url`；生成结果只记录 AI 返回的视频 URL，不自动写入本地大文件。
- 配置弹窗需要验证图片、视频、文本、音频四类默认模型和参数面板能保存、账号同步，并且不会覆盖现有本地多渠道配置。
- 音频独立渠道需要验证全局配置和画布配置节点都能保存 `audioChannelId`，音频生成请求会使用音频模型对应渠道。
- 画布音频节点需要验证工具栏创建、上传、拖入、替换、播放、下载、删除、刷新恢复，以及通过 `/audio/speech` 生成音频节点。
- “我的素材”音频类型需要验证新增、筛选、预览、下载、导入导出、从画布音频节点保存、从素材选择器插入画布、作为视频创作台参考音频以及账号同步。
- 画布图片悬浮工具栏需要验证复制提示词、替换图片、锁定/自由比例、裁剪、局部遮罩编辑、图片放大、反推提示词、查看大图和自定义快捷工具。
- 局部遮罩编辑需要验证点击确认后只创建“局部重绘提示词”和“局部重绘配置”节点，不会自动执行；手动运行配置节点时再通过 `/images/edits` 提交 `mask`，并正确显示耗时和生成结果。
- 局部遮罩编辑失败后重试需要验证仍会带上原始 `mask`，成功后节点状态应从生成中恢复为成功并展示结果。
- 图片拆分工具需要验证从图片节点工具栏打开切图弹窗后，可按行列生成多个图片子节点，并保持正确位置、尺寸和连线。
- 图片放大是浏览器端插值放大，需要验证 1K、2K、4K 目标像素、原图已达到目标时的禁用提示，以及新节点插入位置和连线。
- 反推提示词入口需要验证会从图片节点创建文本节点和文本模式生成配置节点，且自动建立图片、文本到配置节点的连线。
- 画布配置节点上游输入需要验证能统计图片、文本、视频、音频，并能把图片/视频/音频参考组装进 Seedance 视频请求。
- 后端参考素材接口需要验证 `/api/v1/media/references` 上传和 `/api/media/references/:id` 公开访问；配置公网 `PUBLIC_BASE_URL` 后第三方服务应能拉取素材。
- 配置弹窗需要验证每个渠道的“选择模型”按钮会打开和后台管理一致的渠道模型选择弹窗，并能在弹窗内拉取和勾选模型。
- 配置弹窗拉取渠道模型需要验证走后端 `/api/admin/settings/channel-models`，和后台管理“选择渠道模型”一致，不再由前端直接请求渠道 `/models`。
- 配置保存和单独同步 S3/R2 配置后需要验证不再强制刷新页面，当前页面配置应立即生效。
- 失败重试需要验证画布配置节点、生图工作台、视频创作台都能保存重试次数；默认 0 次，接口失败时按配置重试，最终仍失败时展示原错误。
- 生图工作台和视频创作台的生成中、成功和历史卡片需要验证显示实际渠道名称；旧历史没有渠道快照时可不显示。
- 生图工作台和视频创作台底部工作台需要验证桌面宽度下“开始创作”按钮保持在右侧，移动/窄屏下保持可点击。
- “我的素材”文本素材卡片需要验证长提示词默认收起，详情抽屉仍可查看全文。
- 页面右上角需要验证已移除跳转文档按钮，保留配置、主题、版本、GitHub 和账号入口。
- R2/S3 上传需要在网络超时、连接重置或 5xx 时验证 PUT 自动重试，普通 4xx 不应重试。
- 版本更新弹窗需要验证当前本地版本高于远端版本时仍显示本地当前版本，不会把远端旧版本误认为最新。
- 文档体系需要验证 README 和 `docs/index.md` 都指向根目录 Markdown 文档，不再要求启动新文档站。
- Docker 部署需要验证 `DATABASE_DSN=data/infinite-canvas.db` 在存在 `/app/data` 挂载目录时会归一到 `/app/data/infinite-canvas.db`，后台模型配置不会因为工作目录变化读到空库。

## 仍需留意

- WebDAV 同步暂未实装；本次判断为需要先设计账号同步冲突和删除传播规则，不能直接接入。
- AI 超分入口仍是占位提示；当前已完成的是浏览器端图片放大，不是真正的 AI 超分。
