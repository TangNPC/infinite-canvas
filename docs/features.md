# 功能介绍

当前分支是纯生图平台，保留图片、文本、提示词、素材、画布、工作流、账号同步和对象存储能力。视频创作台、音频生成、视频/音频画布节点和相关接口已转移到 `codex/video-audio-upstream-v0.3.3`。

如果需要视频/音频能力，可单独拉取保留分支：

```bash
git clone -b codex/video-audio-upstream-v0.3.3 https://github.com/HuFakai/infinite-canvas.git infinite-canvas-video-audio
cd infinite-canvas-video-audio
cp .env.example .env
docker compose up -d --build
```

## 画布项目

- 支持创建多个画布项目。
- 支持项目重命名、删除、批量选择和批量删除。
- 支持画布项目导入导出，导出包只包含当前纯生图分支需要的图片文件和项目数据。
- 画布项目默认保存在浏览器本地；登录账号并开启同步后，可通过用户配置接口保存画布数据快照。

## 无限画布

- 支持拖动画布、滚轮缩放、缩放滑杆和重置视图。
- 支持小地图定位、点阵/网格线/空白背景、浅色/深色主题。
- 支持框选、多选、全选、取消选择、删除选中、复制粘贴、撤销重做。
- 支持图片节点、文本节点和生成配置节点。
- 支持节点连线，并高亮当前节点相关的上下游节点和连线。

## 图片节点工具

- 上传、拖入、替换、下载图片节点。
- 把图片节点保存到“我的素材”。
- 图片裁剪，并把裁剪结果生成为新的图片节点。
- 图片拆分，按行列生成多个子图片节点。
- 局部遮罩编辑，先生成“局部重绘提示词”和“局部重绘配置”节点，不自动执行。
- 浏览器端图片放大，可按原图比例生成 1K、2K 或 4K 新图片节点。
- 反推提示词节点、查看大图、复制提示词和自定义图片悬浮工具栏。

## AI 生成

项目支持两种 AI 调用方式：

- 本地直连：前端使用本地配置的 Base URL、API Key 和 Model 直接请求 OpenAI 兼容接口。
- 后台渠道：前端请求本项目后端 `/api/v1/*` 代理接口，后端按模型选择管理后台配置的渠道。

当前纯生图分支保留以下 OpenAI 兼容能力：

- `/v1/images/generations`：文生图。
- `/v1/images/edits`：图生图、参考图编辑、局部重绘。
- `/v1/chat/completions`：文本问答和带图问答。
- `/v1/responses`：Responses API 文本/图像相关调用。

模型渠道支持 OpenAI 和 Gemini 两种协议：

- OpenAI 协议保持原有 `/models`、`/images/generations`、`/images/edits`、`/chat/completions` 和 `/responses` 调用结构。
- Gemini 协议会把前端现有请求转换为 Gemini `generateContent` / `streamGenerateContent` 格式，再把返回结果转换回前端现有结构。
- Gemini 支持文本问答、文生图、参考图图生图和 Agent 工具调用。
- Gemini 暂不支持蒙版局部重绘；选择 Gemini 渠道执行带 mask 的局部编辑时会直接返回明确错误，不降级到其他接口。

配置弹窗和后台管理的“拉取模型列表”统一通过后端 `/api/admin/settings/channel-models` 请求渠道 `/models`。如果某个服务不提供 `/models`，需要手动增加模型名。

生图工作台支持：

- 侧边工作台和底部悬浮工作台。
- 多任务并发、失败重试、失败详情、渠道标签、耗时统计。
- 历史结果和实时结果统一展示。
- 结果卡片展示实际图片宽高和文件体积，历史多图结果会展示总文件体积。
- 底部工作台支持常用比例、2K 和 4K 尺寸选项。
- 文本提示词、参考图、本地上传和“我的素材”复用。
- 图片结果下载到本地、保存到“我的素材”和账号历史同步。

画布生成支持停止：

- 配置节点和提示词浮层在生成中会把“开始生成”切换为“停止”。
- 停止会取消浏览器请求，并尽量让后端代理中断上游请求。
- 用户主动停止不会把节点标记为生成失败，节点会恢复为空闲状态。

## Canvas Agent

当前分支合入上游 Canvas Agent 的纯生图子集：

- 新增 `canvas-agent` 本地子项目，提供本机 HTTP/SSE 和 MCP 能力。
- 画布助手的 Agent 模式可以连接本地 Agent，也可以通过网页 Agent Loop 调用当前文本模型渠道。
- 网页 Agent 工具只保留纯生图相关能力：读取画布、读取选区、创建文本节点、创建生图流程、选择节点、触发文本/图片生成。
- 本地 Agent 面板支持手动停止当前 Codex 对话；运行中发送按钮会切换为停止按钮。
- 本地 Agent 对 Codex 空会话和失效 thread 做了兜底，首次发送或旧会话失效时会自动恢复到可用会话。
- 本地 Agent ops 只接受文本、图片和生成配置节点，不包含视频/音频节点或工具。
- WebDAV、视频/音频、Seedance 和上游纯前端化架构未合入当前纯生图分支。

## 创作工作流

- 支持公开/个人工作流模板。
- 支持变量表单、AI 创建工作流、单图/多图系列工作流。
- 支持独立配置提示词文本模型/渠道、图片生成模型/渠道、目标张数、并发上限和审核开关。
- 工作流运行结果进入生图历史。

## 提示词和素材

提示词库支持搜索、标签筛选、详情查看、复制提示词和加入“我的素材”。

“我的素材”当前保留：

- 文本素材。
- 图片素材。
- 按关键词和类型筛选。
- 文本复制、图片预览、图片下载。
- 从提示词库、画布节点、生成结果和服务器素材库加入素材。
- 在画布中插入素材。
- 登录账号并开启同步后，可保存素材数据快照；图片文件以本地 IndexedDB 或对象存储 URL 为准。

服务器素材库支持文本和图片素材的后台管理、前台搜索、筛选、详情查看和加入“我的素材”。

## 会员、支付和登录

合入上游 PR #43 后，当前分支新增：

- 用户端会员中心。
- 用户端我的订单。
- 管理后台会员套餐管理。
- 管理后台订单管理。
- 会员套餐、会员订单和算力点赠送。
- ZPay 聚合支付回调。
- 支付宝当面付扫码支付直连：服务端预创建交易并返回收款二维码内容，前端展示二维码；异步通知验签后校验订单、支付方式和金额。
- 微信支付 APIv3 Native 扫码直连：服务端调用 Native 下单返回 `code_url`，前端展示二维码；异步回调验签解密后校验 appid、mchid、交易类型、订单、金额和币种。
- 通用 OIDC 登录配置。
- 生图排行榜。
- 站点名称、Logo、favicon、描述等站点配置。

## 后端和存储

- Gin 提供 API 服务。
- GORM 管理数据库连接和自动迁移。
- 支持 SQLite、MySQL、PostgreSQL。
- 支持浏览器 IndexedDB、SQLite 元数据和 S3/R2 图片对象存储。
- 支持用户自定义 S3/R2，并可同步模型配置、存储配置、画布数据、生图历史和素材数据快照。
- AI 调用日志按天写入本地 JSONL 文件，后台可查询和清理。
- 业务接口统一返回 `{ code, data, msg }`。

## 当前限制

- 当前分支不包含视频/音频创作能力；需要视频/音频请切换到 `codex/video-audio-upstream-v0.3.3`。
- 画布项目和“我的素材”默认保存在浏览器本地；账号同步保存的是配置和数据快照，不等同于完整云端媒体同步。
- 本地直连模式下，AI API Key 保存在浏览器本地，并由浏览器直接请求配置的 OpenAI 兼容接口；公网多人使用推荐走后台渠道。
- 第三方模型拉取本地参考素材依赖 `PUBLIC_BASE_URL`，如果服务部署在 localhost、内网或不可被第三方服务访问的地址，对方无法拉取参考素材。
- WebDAV 同步尚未实装。
- 真正的 AI 超分尚未实装；当前可用的是浏览器端图片放大。
- 支付宝和微信扫码支付直连仍需要使用真实商户配置、可公网访问的回调地址和生产环境做完整联调。
- 画布更适合桌面端使用，移动端触控体验还未系统完善。
